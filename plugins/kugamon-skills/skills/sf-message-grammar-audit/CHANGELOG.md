# Changelog

## 1.0.0 (2026-04-17)

- Initial release
- Full message extraction pipeline (shared with sf-message-catalog)
- Grammar analysis covering 7 categories: article errors, subject-verb agreement, plural/singular mismatches, spelling, capitalization, punctuation, cross-component inconsistencies
- Three-tier priority system (High / Medium / Low)
- Professional landscape .docx output with summary table and detailed findings
- Reuse mode: can read from existing sf-message-catalog output to skip re-extraction
- Monthly scheduling support
